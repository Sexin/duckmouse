# duck-mouse

## Project setup
```
首先安装node环境https://nodejs.org/zh-cn/
检查node是否安装成功 node -v

安装yarn https://classic.yarnpkg.com/zh-Hans/docs/install/
检查yarn是否安装成功 yarn -v 

安装依赖 根目录执行 yarn add 

若windows用户安装 robotjs失败，先执行npm install --global --production windows-build-tools(墙外下载，速度极慢，耐心等待)
显示all done之后再yarn add 

在根目录下面执行node index.js
```
